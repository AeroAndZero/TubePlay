# TubePlay
 Let your youtube livechat play the game !
## How to Use ?
Run the script and paste the pop-out chat link in the text field and click on read chat.   
To toggle **pause** press '1','2' and '3' simultaneously!

- Tkinter
- Selenium webdriver
